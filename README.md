## ✨ Features

### 🎨 Design & UI
- **Dark Theme** with glassmorphism effects
- **Neon Cyan/Blue Accents** for a modern tech feel
- **Smooth Animations** using AOS (Animate On Scroll)
- **Fully Responsive** for all screen sizes

### 🧭 Navigation
- **Sticky Navbar** with smooth scrolling
- **Hamburger Menu** for mobile devices with slide-down animation
- **Resume Download** with automatic file renaming

### 📄 Sections
- **Hero** with typing animation and social links
- **About Me** with personal bio
- **Technical Skills** as modern pill/tag badges
- **Projects** showcasing WafyOps
- **Education** with academic journey
- **Certifications** with verify links
- **Footer** with all social media links

### ⚡ Additional Features
- **Back to Top Button** (appears after scrolling)
- **SEO Optimized** with meta tags and Open Graph
- **Custom Favicon** for brand identity
